# Damon's Personal Portfolio Welcome to the source code for my personal portfolio website. This project showcases my skills, background, and the various projects I've built as a student and aspiring developer. 
## Overview This website serves as my digital resume and project showcase. It highlights my passion for full-stack development, AI, machine learning, and clean, creative design. 
### Key Sections 
- **Home**: A brief introduction, quick links, and featured focuses.
- **About**: More about me, my background in Computer Science, and my main tech interests.
- **Skills**: A visual representation of my technical skills, including Python, modern JavaScript, and UI/UX design.
- **Projects**: Highlights of my favorite recent works, including:
- **PyWallet**: A personal finance tracking tool.
- **AI Assistant**: A smart ML-powered assistant.
- **TaskTracker**: A browser extension for tracking tasks and deadlines.
- **Contact**: A sleek contact form that lets you reach out to me directly.
## Built With - **HTML5 & Vanilla CSS** for structure and custom styling, including modern fluid layouts. 
- **Vanilla JavaScript** for interactivity, such as feature toggles, mobile menu, and fluid animations.
- **EmailJS** for handling contact form submissions on the frontend without needing a backend server.
- **Font Awesome** for scalable vector icons.
## Features
- **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Light & Dark Mode**: Built-in theme switcher for an accessible user experience.
- **Dynamic Contact Form**: Seamless, secure communication right from the page.
- **Premium UI/UX**: Custom animations, glassmorphism interactions, and modern web design principles.
## Getting Started To run this layout locally: 
1. Clone this repository: ```bash git clone https://github.com/Damon1o/portfolio.git ```
2. Navigate to the project directory: ```bash cd portfolio ```
3. Open `index.html` in any modern web browser to view the site locally.
